  Ces données et le fond de carte viennent du projet collaboratif OpenStreetMap. Cette carte est réalisée par des gens comme vous.
Ces bénévoles ont parcouru les pistes de ski avec leur GPS et ont enregistré leur traces avant de les tracer sur cette carte, d'autres ont utilisé des images satellites.
  Plus de 1'000'000 de contributeurs ont déjà ajouté un petit morceau à cette carte, dont plus de 10'000 pour les pistes de ski.
Pour y faire figurer vos pistes habituelles, c'est facile: Guide du débutant.


Le site

L'utilisation du nom "Opensnowmap.org" fait actuellement l'objet d'une requête pour une licence d'utilisation de la par de la Fondation OpenStreetMap par le biais d'une demande d'antériorité.

  Ce site fait exclusivement appel à des outils open-source. Le code source du site est dispo sur Github. Il est librement réutilisable, sous license GPL pour le code, CC-BY-SA pour le contenu.

Opensnowmap.org est essentiellement le fruit d'une seule personne, et si vous l'appréciez, n'hésitez pas à participer une peu au coût du serveur.


Tile use

  Les tuiles d'opensnowmap.org sont disponibles en deux version, une avec le relief, l'autre avec seulement les pistes de ski. Vous pouvez utiliser les tuiles des pistes seules, sans relief aux conditions suivantes :

  • utiliser un referer valide
  • empêcher le téléchargement en masse
  • crediter Données (c)www.openstreetmap.org & contributeurs ODBL, et www.opensnowmap.org CC-BY-SA.

Le fond de carte topographique et le calque des pistes avec le relief sont trop gourmands en ressources pour être partagés aussi facilement.
