Pistes de ski

   Les pistes de ski d'Opensnowmap.org sont mises à jour quotidiennement.
Dernière mise à jour: **update**

  Ces données et le fond de carte viennent du projet collaboratif OpenStreetMap. Cette carte est réalisée par des gens comme vous.
Ces bénévoles ont parcouru les pistes de ski avec leur GPS et ont enregistré leur traces avant de les tracer sur cette carte, d'autres ont utilisé des images satellites.
  Plus de 350'000 collaborateurs ont déjà contribué, dont 4'000 pour les pistes de ski.
Pour y faire figurer vos pistes habituelles, c'est facile: Guide du débutant.


Le site

  Ce site fait exclusivement appel à des outils open-source. Le code source du site est dispo sur Github. Il est librement réutilisable, sous license GPL pour le code, CC-BY-SA pour le contenu.


Tile use

  Les tuiles d'opensnowmap.org sont disponibles en deux version, une avec le relief, l'autre avec seulement les pistes de ski. Vous pouvez utiliser ces tuiles aux conditions suivantes :

  • utiliser un referer valide
  • empêcher le téléchargement en masse
  • crediter Données (c)www.openstreetmap.org & contributeurs ODBL, et www.opensnowmap.org CC-BY-SA.
