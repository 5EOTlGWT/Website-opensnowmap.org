Donate to Opensnowmap.org

Opensnowmap.org is brought to you freely by Yves Cainaud. The ski map overlay at www.opensnowmap.org is also used freely by other websites and mobile applications.
However, the server rented to run the website have a cost.
In order to support the number of users increase each year, and also with the ultimate goal of being able to render a complete winter-sports map, I humbly ask for your support.
